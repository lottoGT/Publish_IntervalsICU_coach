# 班表與訓練窗口（範例）

如果您有輪班工作、不規律時段、或多重身份要兼顧訓練，可以把「班表 → 訓練窗口」對應寫進 Athlete Context，讓 coach skill 自動依當週班別調整訓練量與時段。

---

## 範例：21 天輪班循環

```
休1 → 晚1 晚2 晚3 晚4 晚5 → 休2 → 中1 中2 中3 中4 中5 →
休1 → 早1 早2 早3 早4 早5 → 休2 → (循環)
```

| 班別 | 工作時段 | 訓練窗口 | 量級調整 |
|------|---------|---------|---------|
| 常日班 | 09:00–17:00 | 全天彈性（早 / 中午 / 下班後）| 標準 100% |
| **晚班** | 22:00–06:00 | 起床後（睡眠優先）| 70–80%、量降強度保 |
| 中班 | 14:00–22:00 | 09:30–13:00（無法進健身房）| 60–70% recovery、家用器材重訓 |
| 早班 | 06:00–14:00 | 15:00–17:30 | 80–90% |
| 休 | — | 全天 | 100%，可排 LSD |

> 請替換為您實際的班別與時段。如果您是固定週末班、夜班護理師、客機機師輪班等，邏輯相同：先列出可訓練窗口長度，再決定量級。

## 訓練類型 × 時段建議

| 訓練 | 標準工作日 | 半天班 | 全休 |
|------|----------|-------|------|
| 高強度（FTP / SS / VO2max） | ✓（下班後體力充足時） | ✓ | ✓ |
| 長騎 / LSD | △（時間不夠） | ✗ | ✓✓ |
| Z2 base | ✓ | ✓ | ✓ |
| 游泳（45–60min） | ✓ | ✓ | ✓ |
| 重訓 | ✓ | ✓ | ✓ |
| 磚訓 brick | △ | ✓ | ✓✓ |
| 賽前 race rehearsal | ✗（時間/體力不夠） | △ | ✓✓ |

## 家用重訓模板（時間/場地受限時）

家用器材：8 / 12 / 16kg 壺鈴、單槓、滾輪、彈力帶。

| 動作 | 組數 | 用具 |
|------|------|------|
| Goblet Squat | 4×8（16kg）| 壺鈴 |
| Single-leg RDL | 3×8/邊（12kg）| 壺鈴 |
| Pull-up | 4×AMRAP | 單槓 |
| KB Swing | 4×15（16kg）| 壺鈴 |
| Turkish Get-up | 3×3/邊（8–12kg）| 壺鈴 |
| Hanging Leg Raise | 3×10 | 單槓 |
| Foam Roll | 10' | 滾輪 |

低週量時不需太多刺激，目標是維持神經肌肉徵召不退化。

## 班表與訓練 Phase 對齊（範例）

聰明的排課會把「低訓練量班別」對齊到「Recovery week」，避免身體雙重壓榨：

| 週次 | 班別 | Phase |
|------|------|-------|
| W1 | 標準 | Test + Re-build |
| W2 | 標準 | Build |
| W3 | 中班 / 低時間班 | **Recovery week** |
| W4 | 早班 | Re-build |
| ... | ... | ... |

---

## Coach Skill 怎麼用這份對應

1. 把當前班表寫進 `~/.endurance-coach/Athlete_Context.md` 或自己的 wiki
2. 開始對話時 coach skill 會自動讀入
3. 排課時 coach 會問「下週是什麼班？」或直接依寫入的對應表決定訓練量

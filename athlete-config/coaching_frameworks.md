## 1. 強度區間（範例：基於 FTP=252w、LTHR=161bpm、跑步閾值 4:08/km、CSS=1:48/100m）

### 統一區間表

| 區間 | 說明 | 單車功率 | 單車心率 | 跑步配速 | 跑步心率 | 游泳配速 |
|------|------|---------|---------|---------|---------|---------|
| Zone 1 | 恢復 | <142w | <109bpm | >5:30/km | <114bpm | >2:10/100m |
| Zone 2 | 有氧低強度 | 142–175w | 109–131bpm | 4:50–5:30/km | 114–131bpm | 2:00–2:10/100m |
| **Zone X** | **⚠️ 中等強度陷阱（嚴禁長時間停留）** | **176–202w** | **132–147bpm** | **4:25–4:50/km** | **132–147bpm** | **1:54–2:00/100m** |
| Zone 3 | 門檻/節奏 | 203–240w | 148–160bpm | 4:08–4:25/km | 148–163bpm | 1:48–1:54/100m |
| Zone 4 | 次最大/最大乳酸 | 241–265w | 161–168bpm | 3:58–4:08/km | 164–170bpm | 1:43–1:48/100m |
| Zone 5 | VO2max | 266–315w | 169–176bpm | <3:58/km | 171–176bpm | <1:43/100m |

**換算公式**：
- 單車功率區間 = FTP × Coggan 7-zone 係數（Z1<55%, Z2 56-75%, Z3 76-90%, Z4 91-105%, Z5 106-120%, Z6 121-150%, Z7>150%）
- 單車 HR 區間 = LTHR × Friel 5-zone 係數
- 跑步配速 = 閾值配速 × 對應百分比
- 游泳區間 = CSS ± 數秒/100m

**VT（通氣閾值）邊界** = 最大心率 75%（範例 ≈ 132 bpm，Zone X 起點）
**低強度上限規則**：心率 < VT，配速比閾值慢 ≥ 20s/km

## 2. 80/20 原則（核心規則）

- **80% 訓練時間在 Zone 1–2（低強度）**，20% 在 Zone 3–5
- **Zone X 是最大陷阱**：感覺「有在練」，實際上消耗恢復能量卻無法帶來低強度訓練的 IL-6 有氧適應，也無法達到高強度訓練的 VO2max 刺激
- 業餘選手最常見錯誤：輕鬆日「稍快但不夠快」——心率落在 132–145bpm 的灰色地帶
- 每週訓練後回顧：低強度時間比例是否 ≥ 80%？若未達標，下週補回

---

## 3. Friel 六大訓練能力（必須識別當季限制因子）

| 能力 | 縮寫 | 說明 | 優先階段 |
|------|------|------|---------|
| 有氧耐力 | AE | 低強度長時間持續 | Base 1–3 |
| 肌肉力量 | MF | 重訓 + 大齒輪低踏頻 | Base 1–2 |
| 速度技術 | SS | 技術動作效率 | 全年 |
| 肌耐力 | ME | 門檻至次最大強度長間歇 | Build 1–2 |
| 有氧能力 | AC | VO2max 短高強度間歇 | Build 2–Peak |
| 衝刺能力 | SP | 最大無氧短衝 | 賽前少量 |

**限制因子（Limiter）識別**：比較 AE/MF/SS（基礎能力）與 ME/AC/SP（進階能力），體能測試顯示哪項最弱即為本季訓練重點。

**重要原則**：基礎能力未穩固前，不做進階能力訓練（AC/SP）。

## 4. 週期化架構

### 訓練週期順序
```
準備期(4週) → 基礎1(4週) → 基礎2(4週) → 基礎3(4週)
→ 建設1(4週) → 建設2(4週) → 巔峰期(2週) → 賽事週 → 過渡期(2–4週)
```

### 各週期能力重點
| 週期 | 主要能力 | 強度特點 |
|------|---------|---------|
| 準備期 | AE + SS | 全低強度；建立訓練習慣 |
| 基礎1–2 | AE + MF + SS | 80/20 嚴格執行；重訓 AA→MT 相 |
| 基礎3 | AE + SS (+ 初期 ME) | 里程衝高；第一個過渡週 |
| 建設1 | ME + AC | Zone 3–4 間歇開始；MF 轉換為 MS |
| 建設2 | ME + AC + SP | 高強度比例增加；比賽特化 |
| 巔峰期 | 維持 ME/AC；加入 SP | 訓練量降低 20–40%；強度維持 |
| 賽事週 | 神經肌肉激活 | 最小量；保留感覺 |

### 恢復週規則
- 每 3 週訓練後安排 1 週恢復（訓練量降 30–40%，強度維持）
- 恢復週指標：疲勞感完全消除，TSB 回正

---

## 5. 周計畫結構（5-2 節奏）

```
週一：恢復日 / 輕鬆游泳
週二：突破課（高強度間歇 or 長距離）
週三：恢復或技術課
週四：突破課（第2個高強度）
週五：恢復
週六：長騎 + Brick
週日：長跑（或長游）
```

**突破課（Breakthrough Workout）** = 本週最重要的課表，需要足夠恢復後才執行，間隔至少 48 小時。

---

## 6. 重量訓練週期

| 階段 | 名稱 | 時機 | 組次 | 強度 |
|------|------|------|------|------|
| AA | 解剖適應 | 準備期 | 3×15–20 | 40–60% 1RM |
| MT | 肌耐力 | 基礎期 | 4–5×12–15 | 60–70% 1RM |
| MS | 最大力量 | 建設期 | 3–5×3–6 | 80–90% 1RM |
| SM | 力量維持 | 賽季中 | 2–3×8–10 | 低量維持 |

---

## 7. 訓練量控制規則

- **10% 規則**：每週訓練量（量、強度或頻率）增幅不超過 10%
- **48 小時窗口**：高強度課後至少 48 小時才能進行下一次高強度
- **長課比例**：長騎不超過週訓練量的 50%；長跑不超過 30%
- **Crash Period**（賽季最多 2 次）：5–7 天訓練量 +50% TSS，之後必須完整恢復週

---

## 8. TSB/體能管理（CTL/ATL/TSB）

| 指標 | 說明 | 目標值 |
|------|------|--------|
| CTL（Chronic Training Load） | 長期體能，42 天加權平均 | 賽季高峰盡量提升 |
| ATL（Acute Training Load） | 短期疲勞，7 天加權平均 | — |
| TSB = CTL - ATL | 疲勞 / 狀態 | 賽前目標 **+10 至 +25** |

**TSB 解讀**：
- TSB < -25：過度訓練風險，需強制恢復
- TSB -10 至 0：正常訓練中，適度疲勞
- TSB +10 至 +25：賽事理想狀態（峰值）
- TSB > +30：過度減量，可能失去訓練感

---

## 9. 恢復警示指標（晨間自評）

每天早上自評下列項目，出現 2 項以上 → 降低當天訓練強度：

- 靜息心率 > 個人基準值 +5（先建立 1–2 週基準）
- 睡眠品質不佳（< 7 小時或夜間醒來）
- 肌肉酸痛或疲勞感 > 3/10
- 情緒低落或對訓練缺乏動力
- 感冒前兆（喉嚨不適、鼻塞）

**過度訓練四階段**：
1. Stage 1（功能性超量，1–2 天恢復）
2. Stage 2（非功能性超量，1–2 週恢復）
3. Stage 3（過度訓練症候群前期，數週恢復）
4. Stage 4（OTS，數月–年）

→ 問題發現越早代價越低；出現 Stage 2 跡象立即減量。

## 10. 賽前 Taper 與賽事配速

### Taper 原則
- Taper 開始前確認 TSB 已在訓練中積累（CTL 夠高）
- 訓練量降低 40–60%，強度維持（關鍵！）
- 賽前 7 天：加入短神經肌肉激活（非 VO2max 高量課）
- 賽前 2–3 天：完全恢復，可做 10–15 分鐘輕鬆游泳

### 各距離目標配速（範例：以上述閾值換算）

**半鐵人 113km**
| 項目 | 目標 | 說明 |
|------|------|------|
| 游泳 1.9km | 2:00–2:05/100m | CSS × ~1.15；輕鬆完成保留能量 |
| 自行車 90km | 65–73% FTP | **切勿超過 79% FTP**；心率 < (LTHR − 13) |
| 跑步 21km | 閾值配速 +27s/km–47s/km | 出 T2 前 5km 保守，心率 < (LTHR − 17)；之後按感覺提速 |

**奧運距離**
| 項目 | 目標 | 說明 |
|------|------|------|
| 游泳 1.5km | CSS × 1.04–1.09 | 接近 CSS，保留體力 |
| 自行車 40km | 91–97% FTP | 高強度但需保留跑步體力 |
| 跑步 10km | 閾值配速 +7–17s/km | 接近但略低於門檻 |

**全鐵人 226km**
| 項目 | 目標 | 說明 |
|------|------|------|
| 游泳 3.8km | CSS × ~1.15 | 慢慢來；心率偏低保留能量 |
| 自行車 180km | 65–72% FTP | 切勿超過 79% FTP |
| 跑步 42km | 出 T2 後心率可能略高；耐心等降至有氧區間再提速 |

## 11. 游泳技術框架（PDLC + 陶米娜框架）

> 整合自：*游泳突破：提速秘訣*（Sheila Taormina）+ Friel PDLC

### 11a. PDLC 基礎框架

| 字母 | 意義 | 關鍵動作 |
|------|------|---------|
| P | Posture（姿勢） | 頭部中立，脊柱對齊，臀部不下沉 |
| D | Direction（方向） | 拉水路徑正確，主導向後推水（不是大S形） |
| L | Length（長度） | 每划伸展最大化，但不靠「滑行」換取划次減少 |
| C | Catch（抓水） | 高肘抓水（EVF），手感為主 |

**踢腿選擇**：
- 2-beat kick：半鐵/全鐵省力選擇
- 6-beat kick：衝刺距離或游泳優先賽事

---

### 11b. 游泳方程式（核心診斷工具）

```
划水次數（次） × 划水周期（秒/次） = 成績（秒）
```

**使用方式**：每次技術調整後，必須同時確認兩個數字——減少了划次但周期拉長，成績反而變慢。

| 常見錯誤 | 後果 |
|---------|------|
| 滑水謬誤（長滑行） | 划次少但周期大幅拉長 → 成績反退 |
| 快速亂划 | 划次多但每次無推進力 → 空轉 |
| 正確目標 | 不靠滑行，減少划次同時維持短周期 |

---

### 11c. 兩大核心技術要素（陶米娜框架）

#### 要素1：高肘早期垂直前臂（EVF — Early Vertical Forearm）

**物理原理**：
- 低肘/直臂 → 力量向下壓水 → 無法前進
- 高肘 → 前臂接近垂直 → 向後推水 → 前進

**正確姿勢要點**：

| 要點 | 說明 |
|------|------|
| 發生時機 | 僅抓水前 1/3，不需全程 |
| 上臂深度 | 低於水面 2.5–10 cm |
| 肘部方向 | 指向上方且略朝外側（不內收） |
| 手腕狀態 | 平直，與前臂平齊（不屈腕）|
| 指尖方向 | 朝向池底 |

**常見錯誤**：

| 錯誤 | 後果 |
|------|------|
| 直臂入水（低肘） | 力量向下，無前進推力 |
| 屈腕 | 水感喪失，誘導低肘 |
| 肘向身體中線收（交叉式）| 背闊肌無法發力 |
| 上臂出水面 | 無法抱水 |

**所需身體條件**：三角肌 + 背闊肌力量（需陸上拉力器訓練建立）；肩部內旋柔韌性。

#### 要素2：水感（Feel for Water）

水感是可訓練的技能，不是天賦。感知要點：
- 手掌和前臂**每個角落**都要有水壓感
- 前臂內側前端（靠指尖）的壓力應比肘窩側更大
- 手指微微張開（一點點），形成邊界層增加感知面積

**心理意象（擇一使用，賽前固定不換）**：
- 簡單版：「用手掌/前臂向後推水，身體向前」
- 進階版：「感受前臂內側的水壓，那是牽引力在作用」

---

### 11d. 拉水三階段

```
入水 → [高肘抓水 1/3] → [斜向階段 1/3] → [完成階段 1/3] → 出水
         肘↑ 前臂↓垂直    上臂斜下45°       手掌緊貼髖部
         指尖朝下          找「靜止的水」     背闊肌+肱三頭肌
```

---

### 11e. 陸上拉力器訓練計劃

> 這是高肘技術建立的**最重要陸上訓練**，水中技術改善的基礎。

**每次三個檢查點**：
1. 指尖朝下
2. 手腕平直
3. 上臂與地面平行（肘與肩同高）

**週程安排（鐵人三項版）**：

| 週次 | 每週頻率 | 每次時長 | 重點 |
|------|---------|---------|------|
| 第1–2週 | 3天 | 7–8分鐘（3×8次/臂） | 慢速，三要點確認 |
| 第3–4週 | 3天 | 7–8分鐘（3×12次/臂） | 加入背闊肌感受，肩旋靠近下巴 |
| 第5週起 | 3–6天 | 7–8分鐘（計時組） | 30s/45s/1min 計時，加入划水節奏 |

**游泳＋拉力器交替訓練（進階，效果最佳）**：
```
6×100m 自由泳，每完成100m 出水做15→30→45→60s 拉力器，休息15秒再繼續
```

---

### 11f. 水中技術訓練

**A. 流線型（最重要的單一練習）**：
- 雙手疊加擠壓，靠近耳朵；身體充分伸展（肋間肌、背闊肌）
- 每次蹬壁保持流線型 4–5 秒（約5–10m）
- 建立肩部柔韌性和高肘所需的核心緊繃狀態

**B. 划水感覺練習（Scull）**：
- 每週3次，6–10×25m
- 高肘姿勢，放慢動作，感受水壓（不求速度）
- 可戴呼吸管，100% 專注水感
- 賽前或主課題前做2分鐘，啟動水感記憶

**C. 技術自我評估（每次下水問自己）**：
1. 抓水時指尖是朝向池底，還是朝前？
2. 手腕是否平直？
3. 疲勞時肘部是否開始下沉？

---

### 11g. 游泳訓練碼（Friel 系統）

- AE1：Zone 1–2 輕鬆；AE2：Zone 2 有氧
- Te1：CSS 門檻訓練
- MF1–3：拉力器 / 阻力道具 / 划臂力量
- SS1–2：技術優先低速練習（Scull 劃水感覺練習即屬此類）
- ME1：Zone 3 門檻長間歇（ME1 = 2×10min CSS）
- ME2：Zone 3–4 中距離間歇
- ME3：Zone 4 次最大強度
- AC1：Zone 5 VO2max 短高強度（60–120s）
- AC2：Zone 5 最大強度爆發

---

### 11h. 鐵人三項游泳特定原則

1. **2次打水**：保留腿力給單車/跑步
2. **開放水域無池壁**：依靠划水節奏感而非蹬壁節奏，練習疲勞後維持周期
3. **賽前啟動水感**：入水後先幾個慢速划水循環，再加速
4. **不靠滑行游泳**：鐵人三項選手容易掉入「身體姿勢至上 + 長滑行」陷阱，參見游泳方程式
5. **CSS配速策略**：半鐵目標 CSS × 1.0–1.03，不超前衝

## 12. 心智訓練原則（Friel Chapter 1）

- **信心（Confidence）**：建立在準備與成功記錄上，而非自我期許
- **心理韌性（Mental Toughness）**：在不舒適中維持高強度的能力
- **耐心（Patience）**：長期計畫中接受短暫退步
- **成功存摺（Success Savings Account）**：記錄每次訓練突破，累積比賽時可提取的心理資本

---

## 13. 賽後過渡期

- 全鐵人後：最少 4 週過渡期（完全輕鬆，無結構訓練）
- 半鐵人後：2–3 週過渡期
- 衝刺距離後：3–5 天恢復

過渡期結束指標：睡眠恢復正常、對訓練重燃動力、晨間心率回基準值。

---

## 14. 訓練計畫等級選擇（Friel）

依個人體能數據與時間預算選擇 Level：
- **半鐵人**：Level 2 或 Level 3（週訓練量 9–13 小時）
- **奧運距離**：Level 2 或 Level 3（週訓練量 8.75–13 小時）
- **衝刺距離**：Level 2 或 Level 3（週訓練量 7.25–12 小時）
- **全鐵人**：Level 2 起步（週訓練量 9.5–16.25 小時，21 週）
